# Web del Club con Sveltekit

## Ejecutar en local

Clonar el repositorio
```sh
git clone https://github.com/mateo-astudillo/cpu-web-svelte.git
```

Moverse a la carpeta
```sh
cd cpu-web-svelte
```

Instalar las dependencias
```sh
npm i
```
